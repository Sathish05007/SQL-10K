Databases Tables CRUD Data Types

1. Creating My First Database

-- Create a database
CREATE DATABASE College;

-- Use the database
USE College;

2. Creating Tables

-- Create a table
CREATE TABLE Students (
    student_id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT,
    course VARCHAR(50)
);

3. Inserting Data

-- Insert single row
INSERT INTO Students (student_id, name, age, course)
VALUES (1, 'Anita', 21, 'Computer Science');

-- Insert multiple rows
INSERT INTO Students (student_id, name, age, course)
VALUES
(2, 'Ravi', 20, 'Mathematics'),
(3, 'Sita', 22, 'Physics');

4. Introduction to SQL Commands

  • DDL (Data Definition Language): CREATE, DROP, ALTER
  • DML (Data Manipulation Language): INSERT, UPDATE, DELETE
  • DQL (Data Query Language): SELECT
  • DCL (Data Control Language): GRANT, REVOKE
  • TCL (Transaction Control Language): COMMIT, ROLLBACK

5. SQL Data Types

Category Examples
Numeric INT, FLOAT, DECIMAL
String CHAR, VARCHAR, TEXT
Date/Time DATE, TIME, DATETIME, TIMESTAMP
Boolean BOOLEAN (true/false)

6. Table Modifications (ALTER)

-- Add a new column
ALTER TABLE Students ADD email VARCHAR(100);

-- Modify datatype
ALTER TABLE Students MODIFY age SMALLINT;

-- Rename a column
ALTER TABLE Students CHANGE course department VARCHAR(50);

-- Delete a column
ALTER TABLE Students DROP email;

7. Updating & Deleting Data

-- Update data
UPDATE Students
SET age = 23
WHERE student_id = 2;

-- Delete a row
DELETE FROM Students
WHERE student_id = 3;

8. Creating & Deleting Databases

-- Create new database
CREATE DATABASE Library;

-- Delete database
DROP DATABASE Library;

9. Creating & Deleting Tables

-- Create table
CREATE TABLE Courses (
    course_id INT PRIMARY KEY,
    course_name VARCHAR(50)
);

-- Delete table
DROP TABLE Courses;
