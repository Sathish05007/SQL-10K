SQL Subqueries

What is a Subquery?

A Subquery (or inner query) is a query inside another SQL query. It is used to fetch results that are then used by the main query.

Syntax

SELECT column_name
FROM table
WHERE column operator (SELECT column FROM table WHERE condition);

Types of Subqueries

  • Single-row Subquery → returns only one row.
  • Multi-row Subquery → returns multiple rows.
  • Nested Subquery → subquery inside another subquery.
  • Correlated Subquery → subquery that depends on outer query values.

Examples

-- Example 1: Students scoring above average
SELECT name, marks
FROM Students
WHERE marks > (SELECT AVG(marks) FROM Students);

-- Example 2: Employees earning more than 'Rahul'
SELECT emp_name, salary
FROM Employees
WHERE salary > (SELECT salary FROM Employees WHERE emp_name = 'Rahul');

-- Example 3: Products with max price
SELECT product_name, price
FROM Products
WHERE price = (SELECT MAX(price) FROM Products);

Practice Questions

  1. Find all students with marks greater than the average.
  2. Find employees earning more than the salary of employee 'Suresh'.
  3. Find the product(s) with the highest price.
  4. List customers whose orders are above the average order amount.
  5. Find employees who earn more than the minimum salary of their department.
